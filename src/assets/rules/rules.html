Classic Mode

Number of players:

min. 6, max. 12;

Roles:

Werewolf, Villager.

Goal of the game:

Werewolf: kill the Villagers; Villager: kill all the Werewolves;

Role assignment:

At the start of the match, roles will be randomly assigned to each player. Based on the number of players, roles will be distributed as follows:

Game phases:

Each phase is timed (Day 120s, Defense 30s, Night 60s).

Day: Discussion and voting. If a player gets >50% votes, they go to Defense.
Defense: The accused pleads. Players vote to save or kill.
Night: Werewolves vote to eliminate a player.

End of the game:

The game ends with a Werewolf victory when the number of living Werewolves is greater than or equal to the number of other players.
The game ends with a Villager victory when all Werewolves have been eliminated.

Advanced Mode

The advanced mode has the same basic mechanics as the classic mode with the addition of new roles (both win with the Villagers):

Roles:

Werewolf, Villager, Doctor, Seer.

Goal of the game:

Werewolf & Villager: same as Classic Mode; Protecctor: during the Night, they choose a player to protect; if that player is chosen by the Werewolves to be eliminated, he is saved and can continue playing; Seer: during the Night, they choose a player and discover whether they are a Werewolf or not;
